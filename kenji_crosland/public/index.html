Welcome to the Amazing Recipe Database!

View all recipes

The database has a simple api. You can make a

  • A GET, and POST request to /recipes to view all recipes or make a new recipe.
  • A PUT or DELETE request to /recipes/:id ot update or delete a recipe.
  • A GET request to /recipes-made-with/:ingredient to view only recipes with a certain ingredient.
  • A PUT request to /recipes/review/:id to add a review to a recipe.
