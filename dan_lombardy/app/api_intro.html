Self Destruct Messenger

Got messages that are sensitive? Do you like more stress in your life? Want to test your memory? Just want to feel like a secret agent for once in your life?

Then use the Self Destruct Messgenger. Post messages for yourself or other viewers. Every time a user views the message, a counter ticks off. Once the counter gets to 4, BOOM no more message.

POST a Message

To create a new message, make a POST request to "api/messages" that contains a JSON object with the following key-values (the values are the value type): 

{"oneWordTitle": String, "oneWordTitle": String, "secretToRead": String, "priority": Number}

GET All Messages

To see all available messages, make a GET request to "api/messages".

NOTE: Viewing a message title in this way does not contribute to or destory a message.

GET Your Message

To view your desired message, make a GET request to "api/{oneWordTitle}" where {oneWordTitle} is the oneWordTitle of the message you would like to view.

NOTE: Viewing a message this way causes a message to either increase count towards or to self destruct. Every time you view a message you will receive a console log that tells you how many views you have left.
